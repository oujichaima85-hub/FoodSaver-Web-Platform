## 📌 Overview
FoodSaver is a full-stack web platform designed to reduce food waste by helping users manage, track, and share food items before they expire.

## ✨ Features
- 🔄 Full CRUD system — Create, Read, Update, Delete food items
- 🔍 Dynamic filtering by category, expiry date and availability
- 📚 Food conservation guides
- 🗄️ Relational database management with MySQL
- 📱 Responsive design

## 🛠️ Tech Stack
| Frontend | Backend | Database |
|----------|---------|----------|
| HTML, CSS, JavaScript | PHP | MySQL |

## 🚀 Installation
1. Clone the repository
```bash
   git clone https://github.com/oujichaima85-hub/FoodSaver-Web-Platform.git
```
2. Import the database — run `database.sql` in phpMyAdmin
3. Configure `config.php` with your database credentials
4. Run with XAMPP or any local PHP server
